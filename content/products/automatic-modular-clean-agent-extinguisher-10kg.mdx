A self-contained, ceiling-mounted extinguisher that needs no detection panel, wiring or external power — the unit activates automatically when ambient heat reaches its trigger point, discharging UL-listed FE-36 clean agent directly onto the protected zone below. Built for switchgear rooms, server racks, sensitive data storage and other unattended spaces that need round-the-clock protection between site visits. Reusable cylinder (not aerosol-based), so it's serviced and recharged rather than replaced.
